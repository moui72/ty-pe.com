# ty-pe.com

Landing page for [ty-pe.com](https://ty-pe.com) — an index of software
[Tyler Peckenpaugh](https://peckenpaugh.us) hosts and builds.

Static single-page site, hosted on GitHub Pages. `index.html` is the whole
thing; `CNAME` binds the apex domain.

Styled entirely from [tylerpeckenpaugh/design-system](https://github.com/moui72/design-system)
tokens and components (inlined, since this stays a single self-contained
file). This is the reference implementation — if a token or component
changes here, mirror the change back into `design-system` before it drifts.
